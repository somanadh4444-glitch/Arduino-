# Arduino-
🔌 Where wires meet code — my Arduino projects, sketches, and circuit experiments.
## 🔧 How to View or Run This Code

### Just want to read the code?
No tools needed! Click on any `.ino` file above — GitHub displays it directly in your browser with syntax highlighting.

### Want to run it on Arduino hardware?
1. Download and install the [Arduino IDE](https://www.arduino.cc/en/software) (free)
2. Click on the project file (`.ino`) → click **"Raw"** → save it (Ctrl+S / Cmd+S), or use the **"Download raw file"** button
3. Open the downloaded `.ino` file in Arduino IDE
4. Connect your Arduino board via USB
5. Select your board: **Tools → Board → [Your Board Name]**
6. Select the correct port: **Tools → Port → [Your Port]**
7. Click **Upload** ⬆️

### Don't have Arduino hardware?
You can still test most sketches virtually using [Wokwi](https://wokwi.com/) — a free online Arduino simulator. Just paste the code in and run it.
